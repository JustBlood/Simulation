# Проект Симуляция
## Суть
В консоли отрисовывается 2d-мир
2d-мир (матрица N*M) состоит из:
1. Существ
    1. Травоядные (едят траву)
    2. Хищники (едят травоядных)
2. Объектов / Ресурсов
    1. Трава
    2. Статичные объекты

Настройки мира:
1. Размер поля
2. Диапазоны HP и скорости существ
3. Диапазон атаки хищников
